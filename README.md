## 📖 About

This quiz covers all **289 official Irish road signs** across every category:

- Regulatory signs (Stop, Yield, speed limits, no-entry, etc.)
- Warning signs (junctions, bends, animals, level crossings, schools)
- Information & motorway signs
- Roadworks signs
- Lane control & overhead gantry signs

Signs are shown in **random order** each time you play. Type the name of the sign, use autocomplete suggestions, and get instant feedback. A 🔥 streak counter keeps you motivated.

## ✨ Features

- 289 signs extracted directly from the official RSA publication
- Autocomplete suggestions as you type
- Smart matching (handles partial answers and minor typos)
- Live score counter (Correct / Wrong / Remaining)
- Progress bar
- 🔥 Streak badge for consecutive correct answers
- Results screen with score percentage
- Download link to the official Rules of the Road PDF
- Works fully offline — no dependencies, no frameworks

---

## 🛠️ Built With

- Python + OpenCV (sign extraction from PDF pages)
- Vanilla HTML, CSS, JavaScript
- No frameworks, no build tools, no server required

---

## 📚 Study Tips

- Use the **autocomplete** to learn the exact official names
- Filter signs by category using the Rules of the Road PDF
- Play multiple rounds — signs shuffle every time
- Focus on warning signs — they're the trickiest category!
